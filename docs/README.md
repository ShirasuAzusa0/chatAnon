# ChatAnon 项目文档

ChatAnon 是一个前后端分离的AI角色扮演聊天应用程序，通过调用大模型能力进行角色扮演。你可以搜索自己感兴趣的角色，并和ta交谈。如果没有喜欢的角色，可以自己创建一个 (ゝ∀･)

## 目录结构

- `frontend/`: 前端相关文件
- `backend/`: 后端相关文件
- `docs/`: 项目文档

## 技术栈

- `前端`：TypeScript、React、Tailwind CSS、shadcn/ui、Zustand、React Router、Live2D
- `后端（旧版，已废弃）`：python3.12、Flask、Flask-RESTful、SQLAlchemy、RSA、JWT
- `后端`：Java 17、Spring Boot 3.5.x、Maven、Spring Security、RESTful API、JPA、RSA、BCrypt、JWT、JDBC、lombok、okhttp、okhttp-sse、fastJson
- `人工智能`：提示词工程、LLM API Key、GPT-SoVITS、TTS
- `数据库`：MySQL、Redis
